安装ArchLinux的简单记录

Warning

对分区表的任何操作都有可能损坏数据，请谨慎操作。

Pre-Installion

先进入ArchISO(这个太简单了，不罗嗦啦)。

验证是否接入有限网。
-
# ping archlinux.org

更新软件时钟。
-
# timedatectl set-ntp true

对磁盘进行分区。
-

格式化新建分区。
-
# mkfs.ext4 /dev/sda1

在其他分区开启SWAP。
-

Installion

将准备安装ArchLinux的分区挂载到/mnt。
-
# mount /dev/sda1 /mnt

安装base base-devel软件包组。
-
# pacstrap /mnt base base-devel vim

Configure

将FSTAB文件注入/etc。
-
# genfstab - U /mnt >> /mnt/etc/fstab

进入已安装的ArchLinux。
-
# arch-chroot /mnt

设置时区。
-
# ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

修改主机名。
-
# hostnamectl set-hostname NAME

更改ROOT用户密码。
-
# passwd

After Installion

安装GRUB。
-

退出CHROOT环境。
-
# exit

解除挂载并重启。
-
# umount -R /mnt && reboot

配置非ROOT用户。
-

安装中文支持。
-
$ sudo pacamn -S wqy-microhei fcitx fcitx-im fcitx-googlepinyin

安装图形界面，显卡驱动，声卡驱动，窗口管理器。
-
$ sudo pacman -S xorg xorg-xinit alsa-utils xf86-video-{nouveau|intel} herbstluftwm

BASHRC & XINITRC。
-

进入图形界面。
-
$ startx

欢迎来到LINUX的世界。
